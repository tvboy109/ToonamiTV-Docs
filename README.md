# ToonamiTV

A private Roku channel for classic animation streaming.

## Installation

1. Visit [my.roku.com/account/add](https://my.roku.com/account/add)
2. Enter channel code: **CJXJPGD**
3. Or click this direct link: [Add ToonamiTV to Roku](https://my.roku.com/account/add/CJXJPGD)
4. The channel will be added to your Roku device

## Features

- Live streaming
- Electronic Program Guide (EPG)
- Classic animation content
- Modern user interface
