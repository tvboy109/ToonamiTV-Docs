# Electronic Program Guide (EPG)

ToonamiTV features a modern, user-friendly Electronic Program Guide that keeps you informed about what's playing now and what's coming up next.

![ToonamiTV Interface](/assets/images/githubtoonamitv.jpg)
*ToonamiTV's modern interface with real-time program guide*

## Mini EPG Features

{: .highlight }
> Quick access overlay showing current and upcoming programming without interrupting your viewing.

### Current Show
- Shows what's playing right now
- Displays accurate time and title
- Easy-to-read format
- Show artwork when available

### Next Up
- See what's playing next
- Countdown to next show
- Instant show preview
- One-click tune-in

### Coming Later
- View up to 3 upcoming shows
- Full schedule timing
- Show titles and times
- Future programming preview

## Schedule Format Example

```json
Current Program:
"12:00 AM FLCL Shoegaze"

Next Program:
"12:30 AM Common Side Effects"

Coming Up:
- "1:00 AM Blue Exorcist"
- "1:30 AM One Piece"
- "2:00 AM One Piece"
```

## EPG Navigation

### Quick Access
1. Press UP during playback to show Mini EPG
2. View current and next shows instantly
3. See upcoming schedule at a glance
4. Press DOWN to hide

### Full Guide
- Access complete program schedule
- Browse future programming
- Show details and artwork
- Easy time slot navigation

## Schedule Updates
- Regular updates from official sources
- Accurate show timings
- Latest programming changes
- Automatic schedule refresh

## EPG Display Options
- Mini overlay during playback
- Full-screen program guide
- Show artwork integration
- Time-based navigation
- Quick show selection
